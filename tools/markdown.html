# 方向

代码开头第一行：graph + 方向

T=Top D/B=Down/Bottom L=Left R=Right

  1. TD/TB 上往下
  2. DT/BT 下往上
  3. LR 左往右
  4. RL 右往左

# 形状

在 vscode 预览需要安装 Markdown Preview Mermaid Support 插件

  • 圆角矩形：表示开始和结束
  • 矩形：表示过程，也就是整个流程的一个环节
  • 单向箭头线段：表示流程进行方向
  • 菱形：表示判断、决策
  • 圆形：表示连接，为避免流程过长或有交叉可将流程切开，圆形即相当于切口处的连接头（成对出现）
  • 另外还有嵌入在以上符号中的描述文本
graph LR;
  A(圆形)-->B[矩形]-->C{菱形}-->D((圆形));
