# Web 开发者包管理器一览

## npm vs yarn

```sh
$ npm init vite-app <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

```sh
$ yarn create vite-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
