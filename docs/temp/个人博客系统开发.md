# 个人博客系统开发

## 项目概述

- 前台可以文章浏览、关键词搜索、登录注册
- 登录后支持对文章进行感谢、评论、点赞
- 后台管理系统可以对文章进行增删改查，分类分页查找；标签管理。

## 技术栈

前端

- Vue 全家桶

后端

- Java 
- Servlet

服务器 

- Tomcat

数据库

- MySQL

数据库名称：MMS （秒木山，修炼）

## 数据库设计

表
- arcticle 文章
- user 用户
- category 分类
