## web 安全

- http 与 https——明文传输与加密传输。
- XSS——跨站脚本攻击，无意中执行恶意代码。
- CSRF——跨站请求伪造，盗用用户身份执行恶意请求。
- 钓鱼网站——仿造高度相似的网站，吸引用户输入敏感信息。
- SQL注入——通过把SQL命令插入到Web表单递交或输入域名或页面请求的查询字符串，最终达到欺骗服务器执行恶意的SQL命令。
- 文件上传/文件包含——上传的文件没进行过滤，例如包含上传包含木马程序的php文件。
- DDoS——分布式拒绝服务攻击。
- postMessage——跨域通信的安全性问题。
- WebStorage——明文存储与加密存储。
- Cross-Origin——同源策略。
- npm——依赖的安全。
- session与认证——会话管理。
- 安全测试工具——简单介绍部分测试工具。
